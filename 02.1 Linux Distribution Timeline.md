## Linux Distribution Timeline

---

**Historical Foundations (1991-1994)**

**1991**
- **Linux Kernel** - Created by Linus Torvalds
  - The foundational core that started it all

**1992 - The Pioneers**
- **SLS** (Softlanding Linux System)
  - One of the first comprehensive distributions
  - Served as a direct ancestor for Slackware and influenced Debian

- **MCC Interim Linux**
  - Early distribution from the University of Manchester
  - Among the very first Linux distributions

- **TAMU Linux**
  - Created at Texas A&M University
  - Notable early distribution with custom tools

- **Yggdrasil Linux**
  - Pioneered the live CD concept
  - One of the first commercially available distributions

**1993 - Foundation of Major Families**
- **Debian** - Founded by Ian Murdock
  - Emphasised community governance and free software
  - Would become one of the most influential base distributions

- **Slackware** - Created by Patrick Volkerding
  - Direct descendant of SLS
  - Oldest Linux distribution still in active maintenance

- **Jurix Linux** - By Florian La Roche
  - German distribution with advanced installation scripts
  - Direct precursor to SUSE Linux

- **Bootroot**
  - Minimalistic distribution running from a floppy disk
  - Demonstrated Linux's flexibility on limited hardware


**1994 - The Enterprise Path Begins**
- **Red Hat Linux** - Founded by Marc Ewing
  - First public release in 1994 (project started in 1993)
  - Established the model for commercial Linux distributions
  - Namesake: Ewing's red Cornell lacrosse hat

---

**Legacy of the Foundation Era**

These early distributions established the core patterns that would define the Linux ecosystem:

  - **Package management** approaches
  - **Installation methodologies**
  - **Community vs. commercial** development models
  - **Specialisation** for different use cases

The four main lineages that would dominate the Linux world: Debian, Red Hat, Slackware, and Arch (later); all trace their philosophical or technical roots to these pioneering distributions from 1991-1994.
