## Creating and Managing Files

A text editor is a software application designed to create, view, and modify plain text files. Unlike word processors, which focus on formatting and layout, text editors work with raw characters without embedded styling information.

---

**Users & Use Cases:**
  - Programmers & Web Developers: Writing source code (e.g., .html,.css,.js,.py,.json,.xml,.md), configuration files (like .env), and many more.

  - System Administrators: Editing configuration files, writing shell scripts, and analyzing logs.
    
  - Writers: Creating content using markup languages (e.g., Markdown, LaTeX) where structure is separated from presentation.

---

  - Creating Empty Files
```bash
# Create a single empty file
touch filename.txt
> filename.txt

# Create multiple files
touch file1.txt file5.txt file9.txt

# Create sequential files
touch f{10..20}.txt          # f1.txt through f10.txt
touch document_{a..c}.txt    # document_a.txt, document_b.txt, document_c.txt
```
> Note: If the file exists, touch updates its modification timestamp.

  - Using cat and Redirection
    - Creates a file with content typed directly or piped in:
```bash
# Create a file with immediate content entry (Ctrl+D to finish)
cat > notes.txt
This is my note.
Ctrl+D

# Append content to an existing file
cat >> notes.txt
Additional note.
Ctrl+D

# Create multi-line file
cat > file.txt << EOF
Line 1
Line 2
Line 3
EOF
```

  - Using echo
    - Creates a file with a single line of content:
```bash
echo "Hello, world!" > myfile.txt      # Create/overwrite file
echo "Another line" >> myfile.txt      # Append to existing file
```

  - GUI Editors

      ```bash
      gedit filename        # Use the Save button or Ctrl+S to save
      ```

    - Mousepad (XFCE)

  - Terminal Based Editors
    
    - nano Editor
      - Simple, beginner friendly
      - Basic Operations
      ```bash
        nano myfile.txt
        # Type content
        # Ctrl+O to (save), Ctrl+X to (exit)
      ```
      - Shortcuts:
        - Basic Operations:
          ```
          Ctrl+O - Write (Save) file
          Ctrl+X - Exit
          Ctrl+R - Read file into current
          Ctrl+K - Cut current line/selection
          Ctrl+U - Uncut (Paste)
          Ctrl+J - Justify paragraph
          Ctrl+C - Show cursor position
          Ctrl+_ - Go to line number
          ```
        - Navigation:
          ```
          Ctrl+F - Forward one character
          Ctrl+B - Backward one character
          Ctrl+Space - Forward one word
          Alt+Space - Backward one word
          Ctrl+P - Previous line
          Ctrl+N - Next line
          Ctrl+A - Beginning of line
          Ctrl+E - End of line
          Ctrl+V - Next page
          Ctrl+Y - Previous page
          ```
        - Search & Replace
          ```
          Ctrl+W - Search
          Alt+W - Find next occurrence
          Ctrl+\ - Search and replace
          ```
        - Editing:
          ```
          Ctrl+D - Delete character under cursor
          Ctrl+H - Delete character before cursor
          Alt+D - Delete word
          Ctrl+Shift+6 - Mark text (start selection)
          Alt+A - Mark text alternative
          Ctrl+^ - Copy selected text
          Alt+T - Cut to end of file
          Ctrl+T - Spell check
          ```
        - Information:
          ```
          Ctrl+G - Display help
          Ctrl+C - Cancel (also shows position)
          ```
        - Advanced
          ```
          Alt+U - Undo
          Alt+E - Redo
          Alt+3 - Comment/uncomment line
          Alt+A - Toggle mouse support
          ```
    
    - vim/vi Editor
      - Powerful, modal editor
      - Modes
        - Normal (press Esc) - navigation & commands
        - Insert (press i, a, o) - text input
        - Visual (press v) - text selection
        - Command line (press :) - ex commands
      - Syntax:
        ```bash
        vi myfile.txt
        # Press 'i' to enter insert mode
        # Write your content
        # Press Esc to exit insert mode
        # :w (save), :q (quit), :wq (save & quit), :q! (quit without saving), :%d (Clear page)
        ```
      - Shortcuts
        - Entering Insert Mode:
          ```
          i - insert before cursor
          I - insert at beginning of line
          a - append after cursor
          A - append at end of line
          o - open new line below
          O - open new line above
          s - delete character and insert
          S - delete line and insert
          ```
        - Navigation
          ```
          h - left
          j - down
          k - up
          l - right
          w - next word
          b - previous word
          e - end of word
          0 - beginning of line
          $ - end of line
          gg - top of file
          G - bottom of file
          :n - go to line n (e.g.,:5)
          Ctrl+f - page down
          Ctrl+b - page up
          Ctrl+d - half page down
          Ctrl+u - half page up
          ```
        - Editing Commands:
          ```
          x - delete character
          dd - delete (cut) line
          dw - delete word
          D - delete to the end of the line
          yy - copy (yank) line
          yw - yank word
          p - paste after the cursor
          P - paste before the cursor
          u - undo
          Ctrl+r - redo
          . - repeat last command
          ```
        - Search & Replace:
          ```
          /text - search forward for "text"
          ?text - search backwards for "text"
          n - next match
          N - previous match
          :s/old/new - replace first "old" with "new" in line
          :s/old/new/g - replace all "old" with "new" in line
          :%s/old/new/g - replace all in file
          :%s/old/new/gc - replace with confirmation
          ```
        - Visual Mode (Selection):
          ```
          v - character visual mode
          V - line visual mode
          Ctrl+v - block visual mode
          y - yank (copy) selection
          d - delete (cut) selection
          > - indent selection
          < - unindent selection
          ```
        - Saving & Exiting:
          ```
          :w - save (write)
          :q - quit
          :wq or ZZ - save and quit
          :q! - quit without saving
          :w! - force save
          :x - save and quit (only if changed)
          :e! - reload file (discard changes)
          ```
        - Buffers & Windows:
          ```
          :sp - split window horizontally
          :vsp - split window vertically
          Ctrl+w w - switch between windows
          Ctrl+w q - close window
          :ls - list buffers
          :bn - next buffer
          :bp - previous buffer
          :bd - delete buffer
          ```
        - Advanced:
          ```
          % - jump to matching bracket
          * - search for word under cursor
          # - search backward for word under cursor
          Ctrl+p - auto-complete (in insert mode)
          :set number - show line numbers
          :set paste - paste mode (disables auto-indent)
          :set nopaste - disable paste mode
          ```

      - Editing:
        ```

        Alt+d - Delete word
        Ctrl+k - Kill (cut) to end of line
        Ctrl+y - Yank (paste)

        ```

  - Hidden Files
```bash
# Create a hidden file (starts with .)
touch .hiddenfile
vim .config_file
```
  - Files with Special Characters
```bash
# INCORRECT (creates two directories):
mkdir my folder

# CORRECT:
mkdir "my folder"          # Using quotes
mkdir my\ folder           # Using escape character

# For files:
vi "my file.txt"
cat "my file.txt"
# Files with special characters
touch "file with spaces.txt"
touch file\ with\ spaces.txt
touch special\*file.txt
touch "project (v2).doc"
```

  - Reading File Content
```bash
cat filename.txt          # View entire file


cat -n filename.txt        # View with line numbers

# Page through long files
less filename.txt      # Press 'q' to quit
more filename.txt

# View beginning/end of files
head filename.txt      # First 10 lines
head -n 20 filename.txt # First 20 lines
tail filename.txt      # Last 10 lines
tail -n 5 filename.txt # Last 5 lines

# Watch log files in real-time
tail -f /var/log/syslog
```

  - Modifying File Content
```bash
# Edit with text editors
vim existing_file.txt
nano existing_file.txt
gedit existing_file.txt

# Append to file without opening editor
echo "New line added" >> existing_file.txt

# Replace entire content
echo "New content" > existing_file.txt
```


> Note: When using vi, if you open a non existent file, it creates a new file with that name.
